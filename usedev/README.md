## 💻 Como Executar o Projeto

1. Clone o repositório:

```bash
git clone <URL_DO_REPOSITORIO>
```

2. Acesse a pasta:

```bash
cd usedev
```

3. Instale as dependências:

```bash
flutter pub get
```

4. Execute o projeto:

```bash
flutter run
```

---

## ⚠️ Observações

* Certifique-se de que os assets estão corretamente configurados no `pubspec.yaml`
* Caso haja erro com imagens, execute:

```bash
flutter clean
flutter pub get
```
